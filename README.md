# Nerix

Nerix is a website security intelligence platform that analyzes websites for security vulnerabilities and provides actionable recommendations.

## Architecture

- Public Web App (React)
- Admin Dashboard (React)
- Backend API (Node.js / Express)
- Security Scan Engine (HTTP, TLS, DNS analysis)

## Features

- Website security scanning
- Security scoring system
- SSL/TLS analysis
- Security headers detection
- DNS security checks
- Admin monitoring dashboard
- User scan history

## Tech Stack

- React
- Node.js
- Express
- PostgreSQL
- Vercel (Frontend hosting)

## Purpose

Built as a portfolio-grade SaaS security intelligence system.
